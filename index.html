O que é Flexbox?

Flexbox (Flexible Box Layout) é um modelo de layout CSS que facilita o alinhamento, a distribuição de espaço e o posicionamento de elementos dentro de um contêiner, mesmo quando o tamanho dos itens é desconhecido ou dinâmico.

Ele ajuda a criar layouts responsivos e flexíveis com menos código e mais controle do que os métodos tradicionais baseados em floats ou posicionamento absoluto.
